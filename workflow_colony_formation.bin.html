workflow_colony_formation.bin.all module¶

This script will run all tasks in a prefect Flow.

When you add steps to you step workflow be sure to add them to the step list and configure their IO in the run function.

run(clean: bool = False, debug: bool = False, **kwargs)[source]¶

Run a flow with your steps.

Parameters

  • clean (bool) – Should the local staging directory be cleaned prior to this run. Default: False (Do not clean)

  • debug (bool) – A debug flag for the developer to use to manipulate how much data runs, how it is processed, etc. Default: False (Do not debug)
